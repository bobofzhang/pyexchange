*pyexchange* - Consistent API wrapper for cryptocurrency exchanges.

## Requirements

 - [requests](http://docs.python-requests.org/en/latest/)

## Installation
```sh
$ pip install -e git+https://github.com/coderiot/pyexchange.git#egg=pyexchange
```
